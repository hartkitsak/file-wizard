# ตัวอย่างการใช้งาน

## Example 1: จัด Downloads ที่รก
**User**: "Downloads ผมมี 500+ ไฟล์ ช่วยจัดเรียงให้หน่อย"

**Process**: วิเคราะห์ → พบเอกสาร, รูป, installers, PDF สุ่ม → เสนอโครงสร้าง Downloads/งาน, ส่วนตัว, Installers, Archive → ถาม confirm → execute → สรุปผล

## Example 2: หาและจัดการไฟล์ซ้ำ
**User**: "หาไฟล์ซ้ำใน Documents ให้หน่อย"

**Output**: แสดงชุดที่พบ (23 ชุด, 156 MB), path ทั้งหมด, ขนาด, date → แนะนำอันที่ควรเก็บ → ถามทีละชุด

## Example 3: จัดโครงสร้าง Projects
**User**: "ช่วยดู Projects folder หน่อย"

วิเคราะห์: โปรเจกต์เก่า 3+ ปีปนกับปัจจุบัน, folder duplicate → เสนอ Projects/Active/{work,side-projects,learning}, Archive/{ปี}, Templates
